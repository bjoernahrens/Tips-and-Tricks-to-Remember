_Magnificent app which corrects your previous console command._

## Installation

Using Homebrew:

```sh
brew install thefuck
```

Add the following to your .bash_profile, .bashrc or .zshrc:

```
eval $(thefuck --alias)
```

and/or

```
eval $(thefuck --alias fix)
```

for the **SFW** version
